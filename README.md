## Highlights

- **Notification Forwarding** - Forward Android notifications to any ntfy server
- **Per-App Filtering** - Choose which apps to forward notifications from
- **Per-Channel Control** - Fine-grained control over notification channels
- **Authenticated Topics** - Support for password-protected ntfy topics
- **Light & Dark Themes** - Material Design theming support
- **Multilingual** - Support for 13 languages
- **Self-hosted** - Works with any ntfy server (public or private)
- **Battery-friendly** - Efficient notification listener service

## Building

```bash
# Debug build
./gradlew assembleDebug

# Release build
./gradlew assembleRelease
```

The APK will be available at `app/build/outputs/apk/`
