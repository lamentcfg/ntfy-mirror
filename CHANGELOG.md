## [1.0.0] - 2024-XX-XX

### Added
- Initial release
- Forward Android notifications to any ntfy server
- Per-app filtering for notification forwarding
- Per-channel control over notification channels
- Support for password-protected ntfy topics
- Material Design theming with light and dark modes
- Support for 13 languages
- Self-hosted server support (public or private)
- Battery-efficient notification listener service
